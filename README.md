# homebrew-airu

airu = ai rules

## Usage

```
brew tap s4na/homebrew-airu
```

```
brew install airu
```

### 初期化

プロジェクトディレクトリで以下のコマンドを実行して、`.airu`ファイルを作成します：

```
airu init
```

### AIルールの設定

`.airu`ファイルを編集して、使用したいAIルールを指定します：

```
# AIルール設定ファイル
# 各行に使用するルールを記述してください
project-rule
command-rule
```

### インストール

AIルールをインストールするには以下のコマンドを実行します：

```
airu install
```

または単に：

```
airu
```

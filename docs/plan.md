# airu install 実装プラン

## 概要
`airu install` コマンドを実行したときに、以下の機能を実装します：

1. カレントディレクトリに `.airu` ファイルが存在するか確認
2. `.airu` ファイルが存在しない場合、ユーザーに `airu init` を実行するよう促す
3. `.airu` ファイルが存在する場合、そこに記載されている AI ルールを読み込む
4. 指定された AI ルールを適切な場所にコピーする
